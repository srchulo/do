=head1 NAME

Data::Object::Base::Regexp

=cut

=head1 ABSTRACT

Data-Object Abstract Regexp Class

=cut

=head1 SYNOPSIS

  package My::Regexp;

  use parent 'Data::Object::Base::Regexp';

=cut

Data::Object::Base::Regexp provides routines for operating on Perl 5 regular
expressions.

=cut
