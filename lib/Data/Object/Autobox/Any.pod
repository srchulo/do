Data-Object Autoboxing for Any Objects

L<Data::Object::Any> objects.

=cut

This package implements the following methods.

=cut

=head2 new

  new(Any $arg1) : AnyObject

Construct a new object.

=over 4

=item new example

  my $any = Data::Object::Autobox::Any->new(\*main);
