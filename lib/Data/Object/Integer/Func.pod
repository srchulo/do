=head1 NAME

Data::Object::Integer::Func

=cut

=head1 ABSTRACT

Functions for Integer Data

=cut

=head1 SYNOPSIS

  use Data::Object::Integer::Func;

=cut

=head1 DESCRIPTION

Data::Object::Integer::Func is an abstract base class for function classes in
the Data::Object::Integer::Func space. This package inherits all behavior from
L<Data::Object::Func>.
